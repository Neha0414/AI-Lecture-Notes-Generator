# AI Lecture Notes Generator

An AI-powered application that converts lecture recordings into study material.

## Features

* Speech-to-Text using Whisper
* AI-generated Notes
* Quiz Generation
* Flashcard Generation
* PDF Notes
* Streamlit Web Interface

## Tech Stack

* Python
* Streamlit
* Whisper
* Ollama
* Gemma 3
* ReportLab

## Workflow

Lecture Audio → Transcript → Notes → Quiz → Flashcards
